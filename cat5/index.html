The Challenge

Wire a straight-through (not crossover) CAT5 ethernet cable so that it tests successfully with a network cable tester.

Documentation

Pinout

Wikipedia's TIA568 page wiring section gives the pinouts for TIA568A and TIA568B configurations. To create a straight through cable, use the same configuration on both ends.

Technique

"Making Ethernet Cables - Tricks of the Trade" has a concise pictoral guide to making a cable.

A YouTube search for "ethernet cable" is handy for watching other people make cables.
