Raspberry Pi Introduction Lab

Objective

Learn how to configure a brand new Raspberry Pi, connect it to a 120V switch (to control a fan, an artisanal light bulb, a blender, etc.), and then control the whole thing with a web browser.

Overview

Equipment

  • Raspberry Pi Model B Revision 2.0 - product page
  • Relay Board - product page
  • Generic USB Wifi Dongle
  • Generic SD Card, at least 4GB
  • Generic Laptop with SD Card Reader
  • Generic Monitor with HDMI
  • Generic USB Keyboard
  • Generic USB Power Supply capable of at least 1A

Software

Note you can find this software on the USB stick for the lab if you're doing it in person.

  • New Out of Box Software (NOOBS) - NOOBS_v1_2_1.zip
  • SD Formatter 4.0 for SD/SDHC/SDXC - SDFormatterv4.zip
